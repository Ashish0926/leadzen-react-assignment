
# Leadzen React + Js Assignment




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/leadzen-react-assignment.git
```

Go to the project directory

```bash
  cd leadzen-react-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
